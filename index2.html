HTTP METHOD

  • GET :-

    • GET method is used to request data from the server from a specific resource. In simple words fetch data from the server.


  • HEAD :-

    • HEAD method is used to ask only information about a document, but not about the document.


  • POST :-

    • POST is used to send data to the server to create/update a resource.


  • PUT :-

    • PUT method is used to update the data in a specific resource.


  • DELETE :-

    • DELETE method is used to request the server to delete a specific resource.


  • OPTIONS :-

    • OPTIONS method is used to check what other HTTP methods can be used for that specific URL


  • TRACE :-

    • TRACE method is used for debugging or diagnostic purposes.
