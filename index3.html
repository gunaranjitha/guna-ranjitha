HTTP STATUS CODES


An HTTP status code is the first line in an HTTP response, that'ssent from a server to the client.

Status codes are expressed through 3-digit numbers, plus a short description.

The first digit of the number identifies the response group.

These responses are grouped in five classes

  • 1xx (100-199) :-

    • Informational responses - indicates the server has received the request and continuing the process.


  • 2xx (200-299) :-

    • Successful responses - indicates the request was successful and the browser has received the information.


  • 3xx (300-399) :-

    • Redirection messages - indicates the user must take additional action to complete the request.


  • 4xx (400-499) :-

    • Client error responses - indicates the website could not be reached due to unavailability of webpage or bad syntax in the request


  • 5xx (500-599) :-

    • Server error responses - indicates the problem was caused by server.
