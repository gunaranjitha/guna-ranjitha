HTTP RESPONSE HEADERS


  • AGE :-

    • The age the object has been in a proxy cache in seconds.

  • CONTENT-DISPOSITION :-

    • Gives information on how to process the response payload.

  • CONTENT-ENCODING :-

    • The type of encoding used on the data.

  • CONTENT-LANGUAGE :-

    • The natural language or languages of the intended audience for the enclosed content.

  • CONTENT-LENGTH :-

    • The length of the response body expressed in 8-bit bytes.

  • CONTENT-TYPE :-

    • Provides the client with the actual content type of the returned content.

  • DATE :-

    • The date and time that the message was sent.

  • ETag :-

    • An identifier for specific version of resource.

  • EXPIRES :-

    • Gives the date/time after which the response expires.

  • LAST-MODIFIED :-

    • The last modified date for the requested object.

  • LOCATION :-

    • Used in redirection, or when a new resource has been created.

  • SERVER :-

    • A name for the server.

  • SET-COOKIE :-

    • Sets an HTTP cookie.

NON-STANDARD HEADERS

  • CONTENT-SECURITY-POLICY :-

    • Helps to protect against XSS (Cross-site scripting) attacks.
